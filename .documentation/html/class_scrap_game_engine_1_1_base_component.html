Detailed Description

Represents a base class for all components attached to a GameObject in the ScrapGameEngine.

The BaseComponent class provides the fundamental interface for all game components, such as rendering, updating, and initialization. It holds a reference to its parent GameObject. Components can be added, updated, and removed as needed throughout the game.

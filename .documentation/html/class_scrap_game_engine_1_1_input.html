Detailed Description

Handles input management for the application, including keyboard and mouse input.

The Input class manages the current and previous state of keys and mouse buttons. It provides methods to query the state of keys, mouse buttons, and mouse position. The input states are updated through the process() method which needs to be called during the application update cycle.

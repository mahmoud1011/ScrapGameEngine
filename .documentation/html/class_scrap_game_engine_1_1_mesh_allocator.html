Detailed Description

Manages the allocation and caching of Mesh objects.

The MeshAllocator class provides static methods to retrieve and cache Mesh objects. It allows efficient sharing of Mesh resources across different parts of the application. Meshes are stored in a cache, identified by a unique key derived from their vertex data.

Mesh * ScrapGameEngine::MeshAllocator::getMesh ( const std::vector< Vertex > & vertices)

Retrieve a Mesh with the given ID or create it if it doesn't exist.

This method either fetches a Mesh from the cache if it has already been created with the same vertex data, or it creates a new Mesh, adds it to the cache, and returns it.

Retrieves a _mesh from the cache based on the provided vertex data.

If a _mesh with the same vertex data is already cached, it is returned. Otherwise, a new _mesh is created, cached, and returned.

Detailed Description

Manages the loading, retrieval, and release of textures.

The TextureAllocator is responsible for handling all aspects of texture management within the game, including loading textures from disk, caching them, and releasing textures when they are no longer needed. It ensures that texture resources are efficiently used and released.

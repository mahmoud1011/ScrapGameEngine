Detailed Description

template<typename... Args>
class Signal< Args >

A signal object that can call multiple slots with the same signature.

A signal object allows you to connect functions to it which will be called when the emit() method is invoked. Any argument passed to emit() will be forwarded to the connected functions.

Template Parameters
ArgsThe argument types that the connected functions expect.
